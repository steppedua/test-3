# Шаблон для тега AI Agent: ассистент по обучению

Из стейта Start по кнопке "Подобрать обучение" можно перейти в стейт AiAgent, в котором происходит работа агента.
По окончании работы агент переходит в стейт Bye, в котором происходит запись полученной информации из сессионной переменной в клиентскую. 

Для корректной работы агента необходимо подключение токена LLM_API_KEY в разделе «Токены и переменные».
Также необходимо наличие аккаунта в сервисе Caila io и пополненный счёт в этом сервисе.

Подробнее с AI Agent можно познакомиться по ссылке: https://help.cloud.just-ai.com/jaicp/dsl/action-tags/built-in-tags/AIAgent/
